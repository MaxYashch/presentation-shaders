Example of working with filters:

On the SLIDE we can see available filter effects in CSS:

1. Grayscale filter - grayscale (controlled). Converts colors in an image to shades of gray
2. Sepia filter - sepia (controlled). Sepia filter change colors with sepia tones, like in old photos.
3. Filter saturation - saturation (controlled). This effect will add color saturation, and make them brighter.
4. Filter hue lighting - hue-rotate (angle). The effect (angle) changes the angle of illumination so that the original image looks with a different color.

5. Filter invert - invert (controlled)
6. This effect flips colors - the amount of effect is controlled
7. Filter opacity - opacity (controlled)
8. Filter brightness - brightness (controlled)
9. Filter contrast - contrast (controlled)
10. Filter blur - blur (radius)
11. Shadow filter drop-shadow (shadow)

Example of setting shader:

#shaded {filter: grayscale (1); transition: filter ease-in 0.3s; }
#shaded: hover {filter: grayscale (0); }
